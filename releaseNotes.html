8.0.0 2023-3-12

This is a major rewrite. The goals are to support: no build step, no Javascript code TrueBases, no delay in new data going live, better testing, and improved community features showing contributions. This release is not recommended. Wait for a few iterations for bugs to get ironed out.

  • üéâ truebase command line app with commands create, delete, list, serve, test and help
  • üéâ Ability to create and run TrueBases using just .truebase settings file.
  • üéâ New dumpStaticSiteCommand for dumping static site since it is not built by default anymore.
  • ‚ö†Ô∏è BREAKING: TrueBaseServer now takes a path to a .truebase settings file. If overriding TrueBaseFolder, that should be the 2nd parameter.
  • ‚ö†Ô∏è BREAKING: No more build step. Instead the site is built in memory only. Use the dumpStaticSiteCommand to get a static site.
  • ‚ö†Ô∏è BREAKING: No more dist folder.
